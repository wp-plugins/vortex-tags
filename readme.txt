== Installation ==

**Required PHP5.**

The Vortex Tags can be installed in 3 easy steps:

1. Unzip "Vortex tags" archive and put all files into a folder like "/wp-content/plugins/votext-tags/"
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Inside the Wordpress admin, go to Options > Vortex Tags, adjust the parameters according to your needs, and save them.
4. Using shortcode [vortex-tags] in post/page content area.
